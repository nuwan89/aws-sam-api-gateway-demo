Simulate object create in bucket

sam local generate-event s3 --bucket poc-devops --key nuwan.txt

Invoke function (s3)

sudo $SAM_HOME local invoke --template s3template.yaml --event hello_world/s3event.json

Package s3 function trigger

sam package --template-file s3template.yaml --output-template-file s3packaged.yaml --s3-bucket glue-poc-thilanka

Create a Cloudformation Stack and deploy s3 trigger

sam deploy --template-file s3packaged.yaml --stack-name sam-s3-app --capabilities CAPABILITY_IAM
